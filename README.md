# Web Screenshot Autobot

본 프로젝트는 특정 사이트의 페이지를 자동으로 탐색하며 전체 화면 스크린샷을 캡처하는 도구입니다.
로그인 세션을 저장하여 로그인 후에만 접근 가능한 페이지도 크롤링할 수 있습니다.

## 📋 필수 요구사항 (Prerequisites)

이 프로그램을 실행하기 위해서는 **파이썬(Python)**이 설치되어 있어야 합니다.

1.  [파이썬 공식 홈페이지](https://www.python.org/downloads/)에서 최신 버전을 다운로드합니다.
2.  설치 프로그램을 실행할 때 반드시 **"Add Python to PATH"** 옵션을 체크해주세요. (이 옵션을 놓치면 실행되지 않습니다.)

## 🚀 시작하기

윈도우 사용자라면 폴더 내의 **`실행.bat`** 파일을 더블 클릭하는 것이 가장 간편합니다.
맥(macOS) 또는 리눅스 사용자라면 터미널에서 **`실행.command`** 파일을 실행하세요.

### 윈도우 (Windows)
1. `실행.bat` 파일을 더블 클릭합니다.

### 맥 / 리눅스 (macOS / Linux)
1. 터미널을 엽니다.
2. 해당 폴더로 이동한 후 다음 명령어를 입력하여 실행 권한을 부여합니다 (최초 1회).
   ```bash
   chmod +x 실행.command
   ```
3. 다음 명령어로 프로그램을 실행합니다.
   ```bash
   ./실행.command
   ```

### 메뉴 설명

1. **환경 구축 (Playwright 설치)**: 최초 1회 실행이 필요합니다. 크롬 브라우저와 필요한 패키지를 자동으로 설치합니다. (파이썬이 설치된 상태에서 진행 가능)
2. **로그인 세션 저장 (로그인 진행)**: 로그인이 필요한 사이트의 경우, 브라우저를 열어 직접 로그인하고 세션(`auth.json`)을 저장합니다.
3. **사이트 크롤링 실행 (스크린샷 저장)**: 입력한 URL을 시작점으로 내부 링크를 탐색하며 스크린샷을 촬영합니다.
    - **최대 페이지 수**: 탐색할 최대 페이지 수를 설정합니다.
    - **대기 시간**: 페이지 로딩 후 스크린샷을 찍기 전 대기할 시간(초)을 설정합니다. (동적 콘텐츠가 많은 사이트에 유용)
4. **결과물 - 스크린샷 폴더 열기**: 촬영된 스크린샷이 저장된 폴더를 탐색기에서 즉시 엽니다.
5. **초기화 (Auth 및 링크 세션 초기화)**: 저장된 로그인 세션과 기존 스크린샷 데이터를 삭제합니다. 다른 페이지를 새롭게 테스트하고 싶을 때 사용하세요.
6. **종료**: 프로그램을 종료합니다.

## 📁 주요 파일 설명

- `실행.bat`: 윈도우용 원클릭 실행 파일 (권장)
- `실행.command`: 맥/리눅스용 실행 파일
- `run.py`: 통합 실행 스크립트
- `save_auth.py`: 로그인 세션 저장 전용 스크립트
- `crawl.py`: 사이트 탐색 및 스크린샷 캡처 핵심 엔진
- `auth.json`: 저장된 로그인 세션 정보 (자동 생성)
- `output/`: 크롤링 결과(스크린샷 및 JSON 데이터)가 저장되는 폴더

## ⚠️ 주의사항

- 크롤링 중 브라우저 창이 보이지 않는 모드(Headless)로 작동하도록 설정되어 있습니다.
- 너무 많은 페이지를 크롤링할 경우 사이트 측에서 차단될 수 있으므로 적절한 `최대 페이지 수`를 설정하세요.
- 스크린샷이 제대로 찍히지 않는 경우 `대기 시간`을 5~10초 정도로 넉넉하게 설정해 보세요.
